Small Preact App Displaying Random Boy or Girl Names

## Installation

```js
cd ./preact-baby-names
yarn install
```

## Build

```js
cd ./preact-baby-names
yarn build
```

## Start

### Development

```js
cd ./preact-baby-names
yarn start
// Open localhost:8000
```

### Production

```js
cd ./preact-baby-names
docker-compose build; docker-compose up
// Open localhost:7000
```

## Test

```js
cd ./preact-baby-names
yarn test
```
